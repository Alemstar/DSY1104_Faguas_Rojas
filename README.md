# DSY1104 - Faguas Rojas

Aplicación web de e-commerce para pastelería con React + Vite.

## Microservicios Backend

- Products BFF: `http://localhost:8282`
- Customers BFF: `http://localhost:8082` (JWT)
- Cart: `http://localhost:8182` (JWT)

## Scripts

```bash
npm install
npm run dev
npm test
npm run build
```
